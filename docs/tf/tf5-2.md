# TF5.2 — Education

Task Force 5.2 creates resources to educate newcomers to perfusion MRI, including tutorial videos, a key-manuscripts bibliography, and a database of example images and artefacts.

The task force aims to create a set of resources to educate newcomers to perfusion MRI. These include a set of tutorial videos describing the physics and applications of perfusion imaging in all its variants. The task force also publishes a bibliography of the key seminal articles for those who wish to learn about perfusion imaging techniques and applications. A further goal is to build a database of examples of perfusion images and image artefacts associated with perfusion.
